# スマプラ勝ち条件（AI導線）- 運用ルール

作成日: 2026-04-10

## ゴール

同じ動画を使った広告で「1日70CV出しているキャンペーンが常に1つはある状態」を維持し、
その数を増やして集客数を拡大する。

## 戦略：当たりCR複製ローテーション

データから判明した構造的事実：
- スパイク（20CV超え日）はキャンペーン寿命の初期に集中（64.5%が0-2日目）
- ピーク翌日は中央値で91%減。連続して30CV超えた例は0件
- 同じCRを複製してもスパイク率は10-33%

→ **1キャンペーンを長く回すのではなく、当たりCRで複数キャンペーンを同時投入し、
初速で当たりを引いたものだけ残す「打席数×選球眼」の戦略が必要。**

## フェーズ1：キャンペーン設定ルール

### RULE-SET-001: キャンペーン構成

| 項目 | 設定 | 根拠 |
|---|---|---|
| CR数 | 6本以上 | 1-2本の30CV超え率1.5%、11-20本は25% |
| ターゲティング | ノンタゲ（AUTOMATIC） | ノンタゲ30CV超え率10.8% vs 手動1.5% |
| ディープファネル | OFF | 30CV超え5件全てOFF |
| 優先アカウント | AI_1, AI_2 | AI_2は30CV超え率15.8%。AI_3/AI_4は0% |

### RULE-SET-002: 使用するCR（素材）の選定

以下の優先順で素材を選ぶ：

1. **過去にスパイク実績のあるCR**（再現率29-33%）
   - 過去の当たりCR、お絵描きムービー等
2. **初日CPA ¥2,000以下を出した実績のあるCR**
   - スパイク群の初日CPA平均 ¥2,065
3. **新規CRで初日にCV出たもの**
   - 初日CV 0のキャンペーンはスパイク率0%

### RULE-SET-003: 同時投入数

- 同一CRで**3-4キャンペーン**を同時に立てる
- 成功率10-33%なので、3本で期待値1本当たり
- アカウントはAI_1とAI_2に分散（ピクセル差の恩恵）

## フェーズ2：早期判定ルール（0-2日目）

### RULE-JUDGE-001: 2日間スクリーニング

配信開始から2日間の合計CVで判定する。これが最も強い予測因子。

| 2日間合計CV | スパイク確率 | 判定 | アクション |
|---|---|---|---|
| ≥20CV | 75% | **即キープ** | 放置。V2に任せる |
| 10-19CV | 33% | **キープ** | 3日目も観察。CPA確認 |
| 3-9CV | 11% | **様子見** | 5日目まで待つ。改善なければ停止 |
| 0-2CV | ~0% | **停止** | このキャンペーンはスパイクしない |

### RULE-JUDGE-002: 初日シグナル

初日だけで見える前兆：

| 指標 | スパイク群 | 非スパイク群 | 判定への使い方 |
|---|---|---|---|
| CV数 | 8.5 | 2.5 | 初日CV=0 → スパイクしない（停止候補） |
| CPA | ¥2,065 | ¥3,605 | 初日CPA ¥2,500以下 → 有望 |
| CPC | ¥188 | ¥290 | CPC ¥200以下 → 有望 |
| 6s視聴率 | 14.32% | 11.23% | 13%以上 → 素材の引きが強い |

**CVRは差がない（10.6% vs 9.8%）。勝敗を分けるのはCVRではなくCPC。
安いクリックを大量に取れる素材＝TikTokのアルゴリズムに好まれる素材が勝つ。**

## フェーズ3：複製ローテーション運用

### RULE-ROTATE-001: 複製サイクル

```
Day 0-1:  当たりCRで3-4キャンペーン同時投入
Day 2:    RULE-JUDGE-001で判定
          → 2日間CV 0-2 → 停止
          → 2日間CV 3-9 → 5日目まで様子見
          → 2日間CV 10+ → キープ
Day 3-5:  キープしたキャンペーンがスパイク（30-85CV日が出る）
Day 5-6:  スパイク後に崩壊（翌日91%減）
          → 即座に次の3-4キャンペーンを投入
繰り返し
```

### RULE-ROTATE-002: 複製タイミング

以下のいずれかで複製を実行：

1. **スパイク翌日**: ピーク後は91%減するので、翌日に次弾を用意
2. **5日目でCV低下**: 2日目判定でキープしたが、5日目に10CV未満に落ちた場合
3. **定期**: キープ中のキャンペーンがあっても、3-5日ごとに予備の複製を投入

### RULE-ROTATE-003: 停止基準

キープ中のキャンペーンを停止する条件：

| 条件 | タイミング | 理由 |
|---|---|---|
| 2日間合計CV ≤ 2 | 2日目 | スパイクしない |
| 3日連続CV = 0 | 随時 | 完全に枯れた |
| CPA > ¥8,064（許容×2） | 随時 | 赤字拡大 |
| CPC < ¥100 | 随時 | 質の低いトラフィック |
| 累計フロントCPO > ¥78,756（許容×2） | 7日目以降 | ファネル全体で利益出ない |
| 累計個別予約CPO > ¥107,590（許容×2） | 14日目以降 | CPA良くても利益出ない |

## フェーズ4：勝ち確定とスケール

### RULE-WIN-001: 勝ち確定条件

直近7日間で**5日以上**、以下の全てを満たすキャンペーンが存在：

1. 日次CV ≥ 18件
2. 当日CPA ≤ ¥4,032（V2が毎回増額する水準）
3. V2到達日予算 ≥ ¥200,000
4. 累計フロントCPO ≤ ¥39,378

→ 現時点では達成例なし。まずは「1日30CV超えが週3日」を中間目標とする。

### RULE-WIN-002: スケール（集客数拡大）

70CV/日キャンペーンが安定したら、同じCRで別アカウントにも展開：

1. AI_1で当たり → AI_2に複製
2. AI_2で当たり → AI_1に複製
3. 2アカウント×70CV = 140CV/日
