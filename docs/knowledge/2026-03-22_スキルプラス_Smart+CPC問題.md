# スキルプラス導線: Smart+配信のCPC問題

## 日付: 2026-03-22

## 結論

スキルプラス導線ではSmart+を使わない方がよい。使う場合はCPC ¥100を監視ラインとする。

## 発見した現象

Smart+キャンペーンでスキルプラス導線の広告を配信すると：

1. **CPCが急激に安くなる**（¥10〜¥30の範囲で推移）
2. しかし安いクリックの**質が低い** → オプトイン率が低下
3. オプトイン率低下 → リストイン率も低下
4. 結果、**リストイン時のCPAが通常配信の2.2倍に悪化**
5. CPA悪化 → 個別予約CPOも連動して悪化

## 因果関係

```
Smart+配信
  → CPC ¥10〜30に急低下
  → 質の低いクリックが増加
  → オプトイン率↓ / リストイン率↓
  → リストインCPA 2.2倍悪化（vs 通常配信）
  → 個別予約CPO悪化
```

## 運用方針

- **スキルプラス導線ではSmart+を使わず通常配信を基本とする**
- もしSmart+を使う場合は**CPC ¥100を下回っていないか常に監視**
- CPC ¥100未満 → 質の低いトラフィック流入の可能性 → 停止検討

## 備考

- AI導線・SNS導線ではこの現象は未確認（スキルプラス特有の問題の可能性）
- スキルプラスはセミナー導線のため、質の高いユーザーの集客が重要
- 安いクリックでセミナーLP流入しても離脱が多い構造
