## 考察

1. **CV減少の主因はSP2の急減速**
   - SP2: 137→18 CV/日（14日で87%減）
   - アクティブ34本中25本がimp=0（配信されていない）
   - 停止ログなし・予算キャップなし → TikTok側の配信制御の可能性

2. **AI_3のスマプラは全滅**
   - 1,070本のスマプラ広告がCPA¥14,517。通常271本がCPA¥3,461
   - スマプラの「AIまとめ」系がCPA¥10,000超で非効率
   - 一方、通常配信の「AI副業の嘘2」（実態はSmart+）がCPA¥2,435で好調

3. **成果の出ているSmart+広告の特徴**
   - AIまとめ（5本入り）: AI_2でCPA¥1,409（CV123）
   - SNSまとめ（9〜20本入り）: SNS3でCPA¥2,049（CV34）
   - 動画数が多い「まとめ」型が安定している

4. **SNS1/SNS2の問題は未解決**
   - SNS1: トークンあり・広告1,205本ENABLEなのにimp=0 → アカウント側の問題
   - SNS2: ¥16万消化でCV0 → ピクセル/イベント設定の問題の可能性
